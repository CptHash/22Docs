Windows Activation
========

The magic command
-----------------

For this, you need to be on the IONIS Wireless Network

Just press Windows + X, choose "Windows PowerShell (Admin)"
In it, copy paste this command:

slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX

And reboot. Your done!
